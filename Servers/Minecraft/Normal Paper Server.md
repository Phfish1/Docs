https://papermc.io/downloads

Download Paper
create a start.bat or start.sh file with: 
```
`java -Xms2G -Xmx2G -jar paper.jar --nogui`
```

Rename paper file to  `paper.jar`
Xms2G and Xmx2g = **RAM** amount

- Agree to eula in `eula.txt`
```
eula=true
```

**start.bat**
