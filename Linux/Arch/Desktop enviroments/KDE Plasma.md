Install Desktop enviroment 
```
$ pacman -S plasma-meta
```

Install KDE applications | LightWeight: `kde-application-meta` **Not Recomended**
```
$ pacman -S kde-applications
```

### KDE Wallet

Might need to install `kdewalletmanager`
```
$ pkill kdewallet
```
