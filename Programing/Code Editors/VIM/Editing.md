### Insert

>i = Edit left 

>I = Edit Far left

### Append

>a = Edit right

>A = Edit Far right

### Opens

>o = Open a line after cursor

>O = Open a line before cursor

9o = Opens 9 lines to write at same time


### Manage 

>x = delete

>r = Replace

### Undo / Redo

>u = Undo

>U = Undo hole line

>CTRL + R = Redo
